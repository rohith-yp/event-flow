# 🚀 EventFlow - Smart Event Booking System

EventFlow is a web-based application that allows users to explore and book events while admins manage events and approvals.

---

## 👤 User Features
- View events
- Book seats
- See available seats
- Low seat warning

---

## 🛠 Admin Features
- Add events
- Delete events
- Approve / Reject bookings

---

## 🏗️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: MySQL
